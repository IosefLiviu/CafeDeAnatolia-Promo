# SCRIPT 03 — "The Countdown / FOMO" (Urgency)

**Duration:** 28 seconds
**Format:** 9:16 (Vertical — Stories, TikTok, Reels)
**Mood:** Fast, urgent, high-energy. "You're running out of time."
**Color Grade:** High contrast, red/amber accents on dark background

## COUNTER ANIMATION SPECS

- Font: Monospace / digital clock style (e.g., "Digital-7", "DSEG7", or similar)
- Color: Starts white → turns amber at 150 → turns red at 50
- Size: Large, center screen, impossible to ignore
- Animation: Numbers cascade/roll like a mechanical counter or slot machine
- Optional: Each number drop triggers a screen shake or flash

---

## SOUND DESIGN

| Element | Description |
|---------|-------------|
| Clock ticking | Mechanical, not digital. Think old station clock. Start at 1 tick/sec, accelerate to 4 ticks/sec |
| Whoosh effects | On each fast cut in the montage sections |
| Bass hit | Single deep 808 hit on the silence-to-poster transition (Shot 8) |
| Music bed | Cafe De Anatolia style track — needs a clear build-up and drop structure |

---

## TEXT STYLING

- All text: UPPERCASE
- Font: Bold sans-serif (Bebas Neue, Oswald Bold, or Impact)
- Animation: Hard slam-in (text appears instantly with screen shake), not smooth fade
- Color: White text, red glow/outline when counter goes red
- Position: Center screen, large enough to read on phone without squinting

---

## PRODUCTION NOTES

- This is the most "TikTok-native" script — fast cuts are essential
- No shot should last more than 3 seconds
- The counter is the hero element — it should dominate the screen
- If real event footage isn't available, use stock: search "club crowd energy", "DJ festival", "concert smoke lasers"
- The silence at 0:20 is critical — that pause creates the psychological hook
- Render at 60fps for smoother fast-cut experience

## VARIATIONS

**Version B — Ad (15s):** Shots 1, 3, 5, 8, 9. Maximum urgency, minimum time.
**Version C — Full (28s):** As scripted above. For Reels/TikTok.
